📌 Problem 1: 704. Binary Search

Binary Search is one of the most fundamental techniques when dealing with sorted arrays. Instead of linearly scanning each element, we divide the array in half at each step, reducing the search time from O(n) to O(log n).


from typing import List

class Solution:
    def search(self, nums: List[int], target: int) -> int:
        if not nums:
            return -1
        if len(nums) == 1 and target in nums:
            return 0 
        l, r = 0, len(nums) - 1
        while l <= r:
            m = (l + r) // 2
            if nums[m] == target:
                return m
            if nums[m] > target:
                r = m - 1
            else:
                l = m + 1
        return -1

✅ What's up:

  • Use Binary Search to optimize search time complexity from O(n) to O(log n).
  • If nums[m] > target, search in left half ([l, m-1]).
  • If nums[m] < target, search in right half ([m+1, r]).
  • The point is always to know your boundaries(l,r) and when to stop (l > r). 😊

📌 Problem 2: 27. Remove Element

At first, I used a simple brute-force traversal method to solve this problem. But after a second thought, I realized the beauty of the two-pointer technique!


from typing import List

class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        l, r, k = 0, 0, 0
        while r < len(nums):
            if nums[r] != val:
                nums[l] = nums[r]
                k += 1
                l += 1
            r += 1
        return k

✅ Key Learnings:

  • Instead of creating a new list, we modify the array in place using two pointers (l and r).
  • Pointer r scans the entire list, while l tracks valid elements.
  • Less complexity? Not really. More elegance? Absolutely. 💅
  • Fun fact: I got a 100% performance score on this! 🎯

📌 Problem 3: 977. Squares of a Sorted Array

Ok, showtime for our beloved friend two-pointer technique when it comes to sorting! 😂


class Solution:
    def sortedSquares(self, nums: List[int]) -> List[int]:
        l, r = 0, len(nums) - 1
        temp = len(nums) - 1
        x, y = nums[l]**2, nums[r]**2
        ans = nums[:]
        while l <= r:
            if x <= y:
                ans[temp] = y
                r -= 1
                y = nums[r]**2
            else:
                ans[temp] = x
                l += 1
                x = nums[l]**2
            temp -= 1
        return ans

✅ Notes:

  • Big applauds: I didn't snap it with sort() and call it a day.
  • How smart is that! Two pointers can go from two ends and end up hugging each other!! 💕
  • Girl now you just lowered a O(nlog(n)) to O(n). 💅
