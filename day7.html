📌 Problem 2: 541. Reverse String II

Seriously, why can't I just copy paste solution for problem 1 and use it as a subfunction?


class Solution:
    def reverseStr(self, s: str, k: int) -> str:
            s = list(s)  # Convert string to list (mutable)
    
            # Reverse a portion of the list
            def reverseSubstring(start: int, end: int):
                while start < end:
                    s[start], s[end] = s[end], s[start]
                    start += 1
                    end -= 1
    
            # Process every 2k block
            for i in range(0, len(s), 2 * k):
                reverseSubstring(i, min(i + k - 1, len(s) - 1))
    
            return "".join(s)  # Convert list back to string

✅ Tricks:

  • Use strating and ending index as the input for the sub function.
  • So it's easier to loop over and reverse every 2k-length string.
