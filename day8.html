ğŸ“Œ Problem 2: Right spin strings

Hey, I know you're thinking about slicing... Trust me, we don't have to apply for extra space.


class Solution:
    def reverseStr(self, s: str, n: int) -> str:
        def reverseSubstring(start: int, end: int):
            while start < end:
                s[start], s[end] = s[end], s[start]
                start += 1
                end -= 1
        s = reverseSubstring(s)
        return reverseSubstring(s[:n]) + reverseSubstring(s[n:])

I'm going through a naive O(m+n)(length of both strings) algo.


class Solution:
    def strStr(self, haystack: str, needle: str) -> int:
        n = len(needle)
        if needle == haystack:return 0
        for i in range(len(haystack)-n+1):
            if haystack[i:i+n] == needle:
                return i
        return -1

ğŸ“Œ Problem 4: 459. Repeated Substring Pattern

Wasn't sure how to start: what substring? what pattern? how to decide? Until I figured out all I need is a dividend larger than 2 and a check of repetitive letters.


class Solution:
        def repeatedSubstringPattern(self, s: str) -> bool:
            n = len(s)
            for i in range(1, n//2+1):
                if n%i ==0:
                    if all([s[j] == s[j - i] for j in range(i,n)]):
                        return True
            return False
